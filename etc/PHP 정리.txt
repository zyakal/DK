<?php   ?> 안에서는 타 언어도 쓸 수있지만 문자열 (" ")로 표현해야한다.

* 변수명 네이밍 규칙

1. 대소문자 영어, 숫자, 특수기호 _ 로 구성
2. 숫자가 맨앞에 오면 안됨
3. 문자 사이에 빈칸이 오면 안됨

$ 는 모두 변수표시
=는 대입연산자

변수 선언후 나중에 값을 넣어줘도 된다
변수는 반드시 = 을 만나야 바뀐다

Literal = 문자그대로 값

문자열을 쓸때는 ' ' , " " 둘다 가능하나 가능하면 " " 가 낫다(변수도 입력이 가능해서)

$n2++  읽기 먼저하고 쓰기가 된다. 즉 바로 프린트하면 반영이 되어있지 않다

++$n2  읽기 전에 쓰기가 되어 바로 반영된다.

// == 비교는 자료형태에 상관없이 안에 있는 값만 비교한다
// === 비교는 모두 비교한다.(권장)

* True , False

문자열이 빈칸이거나 값이 없으면 false, 값이 하나라도 있으면 true
숫자는 0 빼고 다 true

for( 초기화 공간 ; 반복 조건문 ; 증감식 )
while(반복 조건문)


함수 정의할 때 파라미터(매개변수)로 정의한다.
함수 호출할 때 아규먼트(인자) 값을 보내 호출한다.

? 로 시작해서 주소를 불러오는 것을
